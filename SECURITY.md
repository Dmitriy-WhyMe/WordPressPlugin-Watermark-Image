# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 1.x | Yes |

## Reporting a Vulnerability

Please do not disclose security issues publicly before a fix is available.

Report vulnerabilities privately to the maintainer with:

- Affected plugin version
- WordPress and PHP versions
- Steps to reproduce
- Expected impact
- Any proof-of-concept details

You can expect an initial response within 7 days.
