# Watermark Manager (WordPress Plugin)

Плагин для WordPress, который автоматически добавляет водяной знак к изображениям:
- как изображение (PNG),
- или как текст.

Поддерживает выбор типов записей, позицию, отступы, масштаб, прозрачность и массовую генерацию водяных знаков для уже опубликованных материалов.

## Возможности

- Включение и отключение водяного знака.
- Выбор типов записей (`post type`), где применяется обработка.
- 2 режима водяного знака: изображение или текст.
- Позиции: `top-left`, `top-right`, `center`, `bottom-left`, `bottom-right`.
- Настройка масштаба, прозрачности и отступов.
- Автоматическая подмена изображений в контенте и миниатюрах.
- Массовая генерация водяных знаков для существующих материалов.

## Требования

- WordPress 5.8+
- PHP 7.4+
- Расширение GD для PHP

## Сборка ZIP

1. В корне проекта запустите:
   ```bat
   build-plugin.bat
   ```
2. После выполнения появится архив:
   - `dist/watermark-manager.zip`

## Установка (как обычный плагин через ZIP)

1. В админке WordPress откройте `Плагины -> Добавить новый`.
2. Нажмите `Загрузить плагин`.
3. Выберите файл `dist/watermark-manager.zip`.
4. Нажмите `Установить` и затем `Активировать`.
5. Откройте `Настройки -> Watermark Manager` и задайте параметры.

## Использование

1. Включите плагин в настройках.
2. Выберите типы записей, где нужен водяной знак.
3. Выберите тип водяного знака:
- `Image`: загрузите PNG из медиабиблиотеки.
- `Text`: укажите текст, цвет и прозрачность.
4. Выберите позицию, масштаб и отступы.
5. Сохраните настройки.
6. При необходимости запустите массовую генерацию для уже существующих материалов.

## Структура репозитория

- `watermark-manager.php` - основной файл плагина.
- `build-plugin.bat` - скрипт сборки ZIP-архива для установки через WordPress.
