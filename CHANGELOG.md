# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows Semantic Versioning.

## [1.0.0] - 2026-03-27

### Added
- Initial public release of Watermark Manager.
- Image and text watermark modes.
- Post type targeting for watermark application.
- Position, scale, margin, color, and opacity controls.
- Frontend replacement for content and featured image URLs.
- Bulk generation for existing content assets.
- ZIP build script for plugin distribution.
